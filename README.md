# **โปรเจคการวิเคราะห์การซื้อสินค้าออนไลน์ในประเทศเกาหลีใต้ระหว่างปี 2017 ถึง 2024**

## 1. วัตถุประสงค์

      1.1 เพื่อวิเคราะห์แนวโน้มสินค้าและบริการแต่ละประเภท

## 2. ชุดข้อมูล

      2.1 ชุดข้อมูลที่ใช้มาจากหน่วยงานสถิติของประเทศเกาหลีใต้ (KOSIS)
